FMI build project
=================

This directory provides the `CMakeLists.txt.in` template that ships with
source-code FMUs generated by OpenModelica. Importing tools (FMPy,
OMSimulator, ...) can use it to (re)compile the FMU binary from the
sources contained inside the FMU.

CMake is the only supported build system for source-code FMUs. The old
Autoconf+Makefile path has been removed.
